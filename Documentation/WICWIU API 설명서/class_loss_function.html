Detailed Description

template<typename DTYPE>
class LossFunction< DTYPE >

손실 함수를 계산하는 클래스

뉴럴 네트워크의 순전파를 통해 계산된 출력 Tensor와 레이블 값을 비교해 손실 함수를 계산한다.
