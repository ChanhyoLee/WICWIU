Detailed Description

template<typename DTYPE>
class Container< DTYPE >

저장하기 위한 Queue에 해당하는 클래스

Tensor, Operator, Tensorholder 세 가지 클래스에 대한 Queue를 동적으로 할당한다.

기본 queue 구조에 인덱스를 이용한 접근 및 역순으로 접근 등 추가적인 메소드가 구현되어 있다.
