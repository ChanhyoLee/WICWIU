Detailed Description

template<typename DTYPE>
class HingeLoss< DTYPE >

Metric를 이용해 뉴럴 네트워크의 손실 함수를 계산하는 클래스

Hinge Loss 계산 식을 이용해 뉴럴 네트워크의 순전파를 통해 계산된 출력 Tensor와 레이블 값의 손실 함수를 계산한다
