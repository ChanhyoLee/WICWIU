5 ============================논문 버젼 ========================================================
6 reference :
7 https://www.tensorflow.org/api_docs/python/tf/train/RMSPropOptimizer
8 rmsprop algorithm [tieleman2012rmsprop]
9 
10 explain:
11 centered version보다는 Accuracy측면에서는 성능이 떨어지나 속도가 빠름.
12 A detailed description of rmsprop.
13 - maintain a moving (discounted) average of the square of gradients
14 - divide gradient by the root of this average
15 
16 m_aaMeanSqueared = decay * mean_square{t-1} + (1 - decay) * gradient ** 2
17 
18 trainable_data = momentum * trainable_data{t-1} + learning_rate * g_t / sqrt(m_aaMeanSqueared + epsilon)
19 
20 delta = - trainable_data
21 
22 ==========================centered version=======================================================
23 reference :
24 https://www.tensorflow.org/api_docs/python/tf/train/RMSPropOptimizer
25 https://arxiv.org/pdf/1308.0850v5.pdf //논문
26 
27 notice :
28 centered version 사용시 반드시 learning rate는 0.0001로 주어야 한다. 논문참조
29 값이 크면 학습이 잘 안된다.
30 
31 explain:
32  Accuracy 측면에서는 좋으나, 수식이 많아 속도감이 떨어짐.
33 
34 If True, gradients are normalized by the estimated variance of the gradient;
35 if False, by the uncentered second moment. Setting this to True may help with Train,
36 but is slightly more expensive in terms of computation and memory. Defaults to False.
37 This implementation of RMSProp uses plain momentum, not Nesterov momentum.
38 The centered version additionally maintains a moving (discounted) average of the
39 gradients, and uses that average to estimate the variance:
40 
41 mean_grad = decay * mean_square{t-1} + (1-decay) * gradient
42 
43 m_aaMeanSqueared = decay * mean_square{t-1} + (1-decay) * gradient ** 2
44 
45 trainable_data = momentum * mom{t-1} + learning_rate * g_t / sqrt(mean_square - mean_grad**2 + epsilon)
46 
47 delta = - mom
48 
49 ==================================================================================

233  int UpdateParameter(Operator<DTYPE> *pParameter, Tensor<DTYPE> *m_pMeanSquared) {
234  Tensor<DTYPE> *trainable_data = pParameter->GetResult();
235  Tensor<DTYPE> *gradient = pParameter->GetGradient();
236 
237  float signed_learning_rate = this->GetOptimizeDirection() * this->GetLearningRate();//minimizer = -1
238  int capacity = trainable_data->GetCapacity();
239 
240 
241  for (int i = 0; i < capacity; i++) {
242  (*m_pMeanSquared)[i] = (m_decay * (*m_pMeanSquared)[i]) + (( 1.f - m_decay) * ((*gradient)[i] * (*gradient)[i]));
243  (*trainable_data)[i] += ((signed_learning_rate * (*gradient)[i]) / std::sqrt((*m_pMeanSquared)[i] + m_epsilon));
244  }
